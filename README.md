# TFjs WASM Demo
## TensorFlow.js running on WebAssembly backend


To run the project:
1. Clone this repo or download it as zip file
```sh
git clone https://github.com/mrm8488/tfjs_wasm_demo.git
```
2. Go to the folder with the cloned repo
```sh
cd fjs_wasm_demo
```
3. Execute:
```sh
npm i && npm run watch
```
